# 🌍 Every Second on Earth

A real-time live visualization of everything happening on our planet — every single second.

## What it shows

- 👶 **Life & Death** — Births, deaths, marriages, divorces
- ⚡ **Nature** — Lightning strikes, earthquakes, ocean waves
- 🌫️ **Climate** — Trees cut, CO₂ emitted, fish caught, water used
- 📱 **Human Activity** — Google searches, WhatsApp messages, flights in air
- 🚀 **Space** — Stars born, meteors, Earth's journey through space
- 📊 **Since You Opened** — Personal counter since you loaded the page

## Features

- Live counters updating every 100ms
- "Since you opened" personal tracker
- Rotating mind-blowing fact ticker
- World population real-time counter
- One-click Twitter/X share button
- Zero dependencies — pure HTML, CSS & JS

## Data Sources

WHO · United Nations · NASA · WRI · Peer-reviewed research
All numbers based on verified annual statistics divided to per-second rates.

## Tech Stack

HTML • CSS • Vanilla JavaScript
No frameworks. No backend. No tracking.
